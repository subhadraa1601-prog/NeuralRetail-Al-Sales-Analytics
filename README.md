# NeuralRetail – AI Powered Retail Intelligence Platform

## Overview
NeuralRetail is an AI-powered retail intelligence platform designed to analyze retail transaction data and generate actionable business insights. The project combines Data Analytics, Machine Learning, and Business Intelligence techniques to support data-driven decision-making.

## Features
- Customer Segmentation using RFM Analysis
- K-Means Clustering
- Inventory Intelligence & ABC Analysis
- Risk Assessment
- Sales Forecasting
- Business Intelligence Dashboard
- Revenue Analysis
- Customer Retention Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Machine Learning
- Power BI
- Data Visualization

## Dataset Information
- Total Records: 779,425+
- Total Customers: 5,878
- Total Products: 4,631
- Total Revenue: 17.37 Million

## Project Modules
### 1. Data Cleaning & Preprocessing
- Missing value handling

### 3. Inventory Intelligence
- ABC Analysis

- Trend Analysis

### 5. Business Intelligence Dashboard
- Revenue Tracking
- Product Performance Analysis
- Customer Insights

## Key Insights
- Customer retention rate reached 72.39%
- United Kingdom generated highest revenue
- Revenue peaked during October and November
- High-value customer groups identified
